# 🛡️ AntiVirus Guard

برنامج حماية من الفيروسات بسيط وسريع مبني على Python و Kivy.

## ✨ المميزات

- 🔍 فحص شامل للملفات والمجلدات
- ⚡ فحص سريع للملفات المهمة
- 🔐 قائمة كاملة بالأذونات المطلوبة
- 📊 تقارير فصيلة عن الفحوصات
- 🚨 كشف الفيروسات والبرامج الضارة

## 🔒 الأذونات المطلوبة

تطبيق يتطلب 16 أذونة كاملة:

- `READ_EXTERNAL_STORAGE` - قراءة الملفات
- `WRITE_EXTERNAL_STORAGE` - تعديل الملفات
- `CAMERA` - كاميرا الجهاز
- `RECORD_AUDIO` - تسجيل الصوت
- `ACCESS_FINE_LOCATION` - تحديد الموقع الدقيق
- و 11 أذونة أخرى...
